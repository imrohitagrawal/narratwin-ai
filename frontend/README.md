# NarraTwin AI Frontend

This directory contains the Stage 3 frontend foundation for NarraTwin AI.

The scaffold is intentionally minimal:

- Next.js and TypeScript are configured.
- Lint, typecheck, test, and build scripts are available.
- Product workflows remain blocked until the approved Stage 4 slice.

Use the repository-level commands in `docs/LOCAL_DEVELOPMENT.md` and
`scripts/ci/frontend-build.sh` as the source of truth for local validation.
