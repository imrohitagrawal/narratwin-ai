# Skills and Codex Setup

## Recommended order

1. PM Skills: product discovery, strategy, PRD, roadmap.
2. Spec Kit or equivalent: spec/plan/tasks.
3. Addy Osmani Agent Skills: build/test/review/ship lifecycle.
4. TDD skill/workflow: red-green-refactor.
5. Security hardening: uploads, secrets, LLM security, license review.
6. Python testing patterns: FastAPI/provider/RAG tests.
7. Webapp/e2e testing: UI flows.
8. Vercel/Next.js skills: frontend, accessibility, performance.
9. Project doc skills: docs, runbooks, handoff.
10. Wednesday AI agent skills after repo has meaningful code.

## Rule

Do not install every skill blindly. Use the skill needed for the current stage.
