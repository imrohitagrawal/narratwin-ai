# Observability and Cost

## Track per run

- run_id
- project_id
- language
- audience
- depth
- style
- provider
- latency
- cache hit/miss
- token usage when available
- estimated cost when available
- evaluation result
- errors

## Cost controls

- cache scripts
- cache translations
- cache audio metadata
- cache video metadata
- avoid repeated generation unless requested
- default to free engineering mode
