# Product Strategy

## Product thesis

Static GitHub repos, long README files, and English-only demo videos are not enough for global hiring, product onboarding, and technical storytelling.

NarraTwin AI turns approved project knowledge into a multilingual, audience-aware AI avatar walkthrough.

## Target users

- Job seekers building AI portfolios
- Recruiters evaluating technical candidates
- Hiring managers reviewing AI/platform projects
- Engineers who want architecture-level walkthroughs
- Product teams creating onboarding/demo/training material
- Global users who prefer non-English explanations

## Differentiation

The project is not differentiated by "AI avatar" alone.

It is differentiated by:

- grounding in approved project knowledge
- audience-aware explanation
- provider-agnostic architecture
- free engineering mode plus premium showcase mode
- evaluation and observability
- reusable project knowledge pack model

## Positioning

Provider-agnostic multilingual AI avatar walkthrough platform for project demos, onboarding, and technical storytelling.
